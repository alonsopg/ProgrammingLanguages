Puntos Extra.

Apply: Es importante para lisp porque nos ayuda a la tarea de evaluar. Apply junto con eval hacen las evaluaciones 
necesarias para la aplicacion de una funcion. Entonces apply evalua una llamada a una funcion y toma como parametros 
una funcion y una lista de argumentos (apply fn args).

Define Syntax Rules: Es una funcion que nos ayuda a definir macros, es decir nos ayuda extender nuestro lenguaje,
para definir una macro en nuestro lenguaje es necesario pasarle como argumentos los caracteres especiales y la 
definicion de nuestra macro.
